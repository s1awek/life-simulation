### Added
- Initial project structure with Vanilla JS and Vite.
- Neural Network implementation (`src/engine/NeuralNetwork.js`).

- Interactive World and Creature entities (`src/world/`).
- Canvas-based rendering system (`src/renderer/Renderer.js`).
- UI for statistics, charts, and controls (`src/ui/UI.js`).
- Save/Load functionality for simulation state.
- PixiJS renderer (experimental).
